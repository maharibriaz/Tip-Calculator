# Tip Calculator

This project is a simple web-based Tip Calculator built using HTML, CSS, and JavaScript. It allows users to quickly calculate the tip amount based on their bill and selected tip percentage.

## Features

- User can enter the bill amount
- Option to select tip percentage (10%, 15%, 20%)
- Instant calculation with a single button click
- Displays the calculated tip amount clearly
- Simple and clean user interface

## Technologies Used

- HTML – for structure
- CSS – for styling
- JavaScript – for functionality and calculations

## How to Use

1. Open the `tip_calculator.html` file in any web browser.
2. Enter the bill amount in the input field.
3. Select a tip percentage from the dropdown menu.
4. Click the "Calculate Tip" button.
5. The tip amount will be displayed instantly.

## How It Works

The application takes the bill amount and selected tip percentage as input.  
It then uses the formula:

Tip = (Bill Amount × Tip Percentage) ÷ 100

The result is displayed dynamically on the webpage using JavaScript.
